Tugas Cara Install Fedora Linux

Muhammad Naufal Al Bukhori
102230014
IM23B

1. Unduh file ISO Fedora

Mulai dengan mengunduh file ISO Fedora dari situs resmi Fedora. Pilih versi Fedora yang sesuai dengan kebutuhan, seperti Fedora Workstation untuk desktop atau Fedora Server untuk penggunaan server.

2. Membuat virtual machine di VirtualBox

Jika ingin menginstal Fedora di dalam VirtualBox, buat mesin virtual baru dan pilih opsi untuk sistem operasi Linux. Atur jumlah RAM dan penyimpanan sesuai kebutuhan.

3. Memulai Fedora di VirtualBox

Klik “Start” di VirtualBox untuk memulai mesin virtual Fedora. Fedora akan otomatis memproses instalasi dari file ISO.

4. Memulai Proses Instalasi

Pada layar selamat datang Fedora, pilih opsi “Install Fedora” untuk memulai proses instalasi ke komputer atau mesin virtual.

5 . Memilih Bahasa Instalasi

Pilih bahasa yang ingin digunakan selama proses instalasi, lalu klik “Continue.”

6. Pengaturan Waktu dan Tanggal

Di bagian Time & Date, pilih zona waktu Asia dan kota Jakarta, lalu klik “Done” untuk melanjutkan.

7. Memilih jenis penyimpanan

Pada bagian pemilihan penyimpanan, pilih tipe penyimpanan yang sesuai, misalnya ATA VBOX HARDDISK untuk VirtualBox, lalu klik “Done.”

8. Memulai Installasi

Setelah selesai mengatur lokasi, bahasa, keyboard, dan penyimpanan, klik “Begin Installation” untuk memulai proses instalasi Fedora.

9. Memulai Proses Installasi

Tunggu hingga instalasi selesai. Setelah proses instalasi selesai, klik “Finish Installation” untuk menutup jendela instalasi.

10. Selesai

Fedora sudah terinstal dan siap digunakan. Reboot atau mulai Fedora dari VirtualBox untuk menggunakan sistem operasi baru.
